# Sirius.Brokerage 

Sirius blockchains brokerage service. 
Manages broker accounts and accounts associated with them. 
Detects incoming transfers and allows to execute outgoing transfers.

Owns models:

- Broker account
- Account
- Deposit
- Withdrawal
